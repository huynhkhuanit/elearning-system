📱 RESPONSIVE DESIGN

MOBILE (<640px):

  ├─ Content:       Full width
  ├─ Layout:        Single column
  ├─ Navigation:    Bottom or hamburger menu (optional)
  └─ Cards:         Stacked vertically

TABLET (640-1024px):
  ├─ Sidebar:       Toggleable (80px collapsed default)
  ├─ Content:       Adjustable width
  ├─ Layout:        2-column grids
  ├─ Stats:         2 columns per row
  └─ Cards:         Responsive gap

DESKTOP (>1024px):
  ├─ Sidebar:       Always visible (256px)
  ├─ Content:       Main area with margin
  ├─ Layout:        3-4 column grids
  ├─ Stats:         4 columns per row
  └─ Cards:         Optimal spacing


🔐 SECURITY & AUTHORIZATION
─────────────────────────────────────────────────────────────────────────────

AUTHENTICATION FLOW:
  1. User visits /admin
  2. LayoutWrapper detects /admin/* path
  3. AdminLayout checks user role
  4. If role = admin || teacher → Show dashboard
  5. Else → Show access denied page

ROLE REQUIREMENTS:
  ✓ Admin        - Full access
  ✓ Teacher      - Full access
  ✗ Student      - No access (denied page)
  ✗ Unauthenticated - Redirect to login

PROTECTED ROUTES:
  /admin/*              - Role check
  /api/admin/courses-full - Server-side validation (existing)

🚀 HOW TO USE
─────────────────────────────────────────────────────────────────────────────

1. LOGIN AS ADMIN/TEACHER:
   - Visit application
   - Sign up or login
   - Account must have role: 'admin' or 'teacher'

2. ACCESS ADMIN DASHBOARD:
   - Go to /admin
   - Dashboard loads with stats and quick actions

3. NAVIGATE:
   - Sidebar: Click items to navigate
   - Desktop: Sidebar always visible
   - Mobile: Toggle with menu button

4. MANAGE CONTENT:
   - Go to /admin/lessons
   - Browse courses, chapters, lessons
   - Click lesson to edit markdown content

5. CONFIGURE SETTINGS:
   - Go to /admin/settings
   - Update system configuration
   - Click "Save Settings"
